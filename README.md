# solivram — Releases officielles

Releases officielles de solivram : paquets `.deb`, binaires, changelogs.

> Infrastructure distribuée haute disponibilité, sécurisée post-quantique,
> conçue en Rust pur pour des environnements critiques.

## Installation (Debian / Ubuntu)

```bash
sudo dpkg -i solivram_0.1.0_amd64.deb
solivram --help
```

## Vérification de signature

```bash
solivram identity:verify
# ✅ P-256 valide | ✅ ML-DSA valide
```

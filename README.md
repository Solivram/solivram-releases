## Installation (Debian / Ubuntu)

```bash
sudo dpkg -i solivram_0.1.0_amd64.deb
solivram --help
```

## Vérification de signature

```bash
solivram identity:verify
# ✅ P-256 valide | ✅ ML-DSA valide
```
